# Telegram News Bot

This is a Telegram bot that fetches the latest news and sends it directly to your chat.

## Features

- Fetches the latest news from NewsAPI.
- Responds to the `/news` command in Telegram.

## Setup

### Prerequisites

- Python 3.x
- Telegram Bot Token
- NewsAPI Key
